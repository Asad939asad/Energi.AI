**Energi.AI** is a modern, responsive HTML + Tailwind CSS energy insights dashboard.  
It combines beautiful UI with smart logic to visualize energy predictions and recommendations dynamically.

---

## 🛠️ Tech Stack

- [**Vite**](https://vitejs.dev/) — Lightning-fast front-end tooling
- [**Tailwind CSS**](https://tailwindcss.com/) — Utility-first CSS framework
- [**PostCSS**](https://postcss.org/) — CSS transformer
- [**Flask**](https://flask.palletsprojects.com/) — Lightweight Python backend
- [**Gemini AI**](https://ai.google.dev/) — Google Generative AI for insights



## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/Asad939asad/Energi.AI.git
cd Energi.AI

# 2. Install frontend dependencies
npm install

# 3. Start the development server
npm run dev

# 4. (Optional) Run Flask backend in another terminal
python app.py
